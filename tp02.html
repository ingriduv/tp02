1- ¿que es un url?

es una cadena de texto que especifica dónde puede encontrarse un recurso (página, imagen, video, sonido, etc.)

2- ¿ q es una url absoluta?

Las URL absolutas proporcionan la URL completa del documento como el nombre de dominio y la ruta como necesario

3- ¿qué es una url relativa?

Las URL relativas describen la ruta a un archivo relativo al actual documento se pueden usar cuando se vincula a otro documento en su propio sitio

4- ¿cuál es la diferencia entre una url relativa y una url absoluta?

Una URL absoluta incluye todas las partes de la URL, mientras que la URL relativa incluye solo algunas con la finalidad de hacerla más breve

6- ¿qué es un directorio? ¿qué relación hay con una carpeta?

es un tipo exclusivo de archivo que sólo contiene la información necesaria para acceder a archivos o a otros directorios. En consecuencia, un directorio ocupa menos espacio que otros tipos de archivos

7- En la estructura de un proyecto web, ¿a qué se llama directorio raíz? ¿Con qué símbolo se representa?

El directorio raíz o root directory es el directorio principal que, en los sistemas operativos parecidos a Unix, contiene todos los demás directorios y archivos, lo cuales operan con lo que se conoce como un sistema jerárquico y se designan con una barra diagonal y/o inclinada(/)

10- ¿Cuál es el propósito de la etiqueta anchor?

Nos permite crear un enlace (hipervínculo) a otra página web y/o un recurso ubicado en el mismo proyecto u en otro servidor web

11- ¿Qué atributo se utiliza en la etiqueta anchor para especificar la URL a la que debe enlazar?

El atributo href de un enlace indica la relación entre páginas a los motores de búsqueda
